===============
How to use this
===============


CLI: Interactive mode
---------------------


Investigate all folder files
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: console

    $ pyastrx -i -d src

Investigate a single file
~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: console

    $ pyastrx -i -f src/module.py


As a linter
-----------


.. code-block:: console

    $ pyastrx -l -d src
